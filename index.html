CSS

Ո՞րն է գլխավոր տարբերությունը "display: block;" և "inline;"-ի միջև:

block-ը սկսվում է նոր տողից և զբաղեցնում ողջ լայնությունը (div, p)։ inline-ը չի սկսվում նոր տողից, զբաղեցնում է միայն իր տեքստի չափով տեղ (span, a)։

CSS

Ի՞նչ է տալիս "display: inline-block;" հատկությունը:

Էլեմենտը մնում է տողի մեջ (ինչպես inline-ը), բայց նրա վրա սկսում են աշխատել չափսերը՝ width, height, margin և padding (ինչպես block-ը)։

CSS

Ինչո՞վ է տարբերվում "display: none;"-ը "visibility: hidden;"-ից:

display: none-ը լիովին ջնջում է էլեմենտը էջից (տեղ չի զբաղեցնում)։ visibility: hidden-ը թաքցնում է տարրը, բայց նրա դատարկ տեղը էջում մնում է։

CSS

Ի՞նչ է CSS Grid-ը (display: grid;):

Երկչափ (2D) դասավորության համակարգ է, որը թույլ է տալիս էջի տարածքը բաժանել ճկուն տողերի և սյունակների (ցանցի)։

HTML

Ի՞նչ է անում input-ի "placeholder" ատրիբուտը:

Ցույց է տալիս թույլ մոխրագույն հուշող տեքստ դաշտի ներսում, որն անհետանում է, երբ օգտատերը սկսում է որևէ բան տպել։

HTML

Ի՞նչ է անում input-ի "value" ատրիբուտը:

Սահմանում է ներմուծման դաշտի սկզբնական (կանխադրված) արժեքը, որը երևում է էջը բացելիս։

HTML

Ի՞նչ է անում "disabled" ատրիբուտը input-ի մեջ:

Արգելափակում է դաշտը կամ կոճակը. օգտատերը չի կարողանում սեղմել դրա վրա կամ տեքստ փոփոխել։

HOSTING

Պարզ խոսքերով՝ ի՞նչ է Հոսթինգը (Hosting):

Դա մի հզոր համակարգիչ է (սերվեր), որը միացված է համացանցին 24/7, և որտեղ վարձակալում ենք տարածք՝ մեր կայքի ֆայլերը պահելու համար։

DOMAIN

Պարզ խոսքերով՝ ի՞նչ է Դոմենը (Domain):

Դա կայքի անվանումն է կամ հասցեն (օր.՝ google.com), որը մարդիկ հավաքում են բրաուզերում՝ կայքը գտնելու համար։

CSS FLEX

Ի՞նչ է անում "flex-direction" հատկությունը:

Որոշում է flex-տարրերի դասավորության ուղղությունը. row (հորիզոնական տողով՝ լռելյայն) կամ column (ուղղահայաց սյունակով)։

CSS FLEX/GRID

Ի՞նչ է անում "gap" հատկությունը:

Սահմանում է հստակ և հավասար հեռավորություն (բացատ) flex կամ grid կոնտեյների ներսի էլեմենտների միջև։

CSS GRID

Ի՞նչ է սահմանում "grid-template-columns"-ը:

Որոշում է grid ցանցի սյունակների քանակն ու չափսերը (օր.՝ `repeat(3, 1fr)` ստեղծում է 3 հավասար սյունակ)։

CSS GRID

Ի՞նչ է անում "grid-column: span 2;" հրամանը:

Grid-ի ներսի էլեմենտին ստիպում է տարածվել և զբաղեցնել 2 սյունակի տեղ։

CSS ALIGN

Ինչպե՞ս կենտրոնացնել տեքստը բլոկի ներսում:

Օգտագործելով CSS-ի text-align: center; հատկությունը։ Այն աշխատում է տեքստի և inline տարրերի վրա։

CSS ALIGN

Ինչպե՞ս կենտրոնացնել բլոկային (block) էլեմենտը:

Էլեմենտին պետք է տալ ֆիքսված լայնություն (width) և կիրառել margin: 0 auto; հատկությունը։

CSS ALIGN

Ինչպե՞ս Flexbox-ով էլեմենտը բերել կատարյալ կենտրոն (X և Y առանցքներով):

Ծնող տարրին պետք է տալ՝
display: flex;
justify-content: center;

CSS ALIGN

Կիրառելով CSS-ի text-align: right; հատկությունը։

CSS ALIGN

Ինչպե՞ս առանձին բլոկային (block) էլեմենտը տանել դեպի ԱՋ եզր:

Կիրառելով margin-left: auto; հատկությունը։ Այն ձախ կողմից առավելագույն ազատ տարածություն է ստեղծում և հրում բլոկը աջ։

CSS FLEX

Ինչպե՞ս Flexbox-ում բոլոր տարրերը տանել դեպի ԱՋ:

HTML IMAGE

HTML IMAGE

Ինչո՞ւ է պարտադիր նկարի "alt" ատրիբուտը:

Այն պարունակում է նկարի տեքստային նկարագրությունը, որը երևում է, եթե նկարը չբեռնվի, օգնում է տեսողական խնդիրներ ունեցողներին և SEO-ին։

Որո՞նք են հիմնական իմաստային (semantic) թեգերը կայքի կառուցվածքի համար:

<header> (գլխամաս), <nav> (նավիգացիա), <main> (հիմնական բովանդակություն), <footer> (կայքի ներքևի մաս)։

display: none-ը լիովին ջնջում է էլեմենտը էջի կառուցվածքից (տեղ չի զբաղեցնում)։ visibility: hidden-ը թաքցնում է տարրը, բայց նրա դատարկ տեղը էջում մնում է անփոփոխ։
